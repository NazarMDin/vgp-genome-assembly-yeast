# Results

This folder contains output files and screenshots from the Galaxy tutorial run.

## How to populate this folder

After completing the tutorial in Galaxy, download and place the following files here:

### Text outputs (copy from Galaxy dataset previews)

| File | Source in Galaxy |
|---|---|
| `genomescope_summary.txt` | GenomeScope2 → Summary output |
| `gfastats_hap1_hap2.txt` | gfastats on hap1 and hap2 contigs |
| `busco_summary_hap1.txt` | BUSCO hap1 → Short summary text |
| `busco_summary_hap2.txt` | BUSCO hap2 → Short summary text |
| `merqury_qv_stats.txt` | Merqury → QV stats |
| `merqury_completeness_stats.txt` | Merqury → Stats collection |
| `bionano_stats.txt` | Bionano gfastats output |

### Images (download as PNG/JPG)

Place downloaded images in the `screenshots/` subfolder:

| File | Source |
|---|---|
| `screenshots/genomescope_linear_plot.png` | GenomeScope2 linear plot |
| `screenshots/genomescope_log_plot.png` | GenomeScope2 log plot |
| `screenshots/busco_hap1.png` | BUSCO summary image hap1 |
| `screenshots/busco_hap2.png` | BUSCO summary image hap2 |
| `screenshots/merqury_spectra_cn.png` | Merqury CN spectrum plot |
| `screenshots/merqury_spectra_asm.png` | Merqury ASM spectrum plot |
| `screenshots/pretext_pre_yahs.png` | Hi-C contact map before YaHS |
| `screenshots/pretext_post_yahs.png` | Hi-C contact map after YaHS |
